## Prerequisites

- yarn
- node

## Backend

The project's backend was developed in NodeJS.

### Steps to run

1. Go to the *backend* folder
2. Install dependencies:
    ```
    yarn
    ```
3. Run project: 
    ```
    yarn dev:server
    ```

## Frontend

The project's frontend was developed in ReactJS.

### Steps to run

1. Go to the *frontend* folder
2. Install dependencies:
    ```
    yarn
    ```
3. Run application: 
    ```
    yarn start
    ```
